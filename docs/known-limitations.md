# Known Limitations

This project is in MVP stage. Current limitations include:

- ❌ No authentication yet integrated (e.g. GOV One Login or Staff Auth)
- ⚠️ Some pages are static or use placeholder content
- ⚠️ Layouts are evolving — staff layout is a placeholder only
- ❌ i18n not yet implemented — only English is supported
- ❌ No production-grade form validation or accessibility testing yet

These will be iterated on during Alpha/Beta phases.
